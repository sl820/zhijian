## Feistel 实现细节（PoC 验证）

```
F(x, k): xorshift-multiply 链
- h = x ⊕ k
- h = imul(h, 0x85ebca6b)
- h ⊕= h >>> 13
- h = imul(h, 0xc2b2ae35)
- h ⊕= h >>> 16
```

4 轮 Feistel + 16-bit halves：
- `feistel(x)`: 4 轮 (L, R) → (R, L ⊕ F(R, k_i))
- `feistelInv(x)`: keys 倒序 + swap 方向反
- `feistelInv(feistel(x)) === x` (测试 0..999 全过)
- 10000 samples 16 桶分布 < 1.5x 均匀（chi-square 粗检）
